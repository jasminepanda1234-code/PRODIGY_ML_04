# Task 4 - Hand Gesture Recognition

## Objective
Classify 10 different hand gestures using 
Support Vector Machine (SVM).

## Dataset
Leap Gesture Recognition (Kaggle)
- 20,000 images
- 10 gesture classes

## Model
- Algorithm: SVM (RBF Kernel)
- Accuracy: 99.80% 🔥
- Images: 64x64 Grayscale

## Gesture Classes
01_palm, 02_l, 03_fist, 04_fist_moved,
05_thumb, 06_index, 07_ok, 
08_palm_moved, 09_c, 10_down

## Libraries Used
- Python, NumPy, OpenCV
- Scikit-learn, Matplotlib# PRODIGY_ML_04
